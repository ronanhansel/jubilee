# Instructions
## Environment:
`Python >= 3.8`

## Set up

install dependencies
```
  >>> pip install discord
  >>> pip install DiscordUtils[voice]
```
See `requirements.txt` for more details

## Environment variables
### For bot
Key | Variable
------------- | -------------
discord_token | token for discord bot
